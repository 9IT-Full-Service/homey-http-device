HTTP Device adds a virtual device with 8 configurable buttons to your Homey. Use these buttons as flow triggers to send HTTP requests (GET, POST, PUT, DELETE, PATCH) to any URL. Each flow lets you dynamically specify the target URL, HTTP method, request body, and custom headers.

This app is ideal for integrating external APIs, webhooks, and smart home services that are not natively supported by Homey. Simply add the virtual HTTP Device, create a flow with a button trigger, and configure the HTTP request action with your desired endpoint and parameters.
